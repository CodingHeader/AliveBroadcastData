### 2.2 后台布局
后台采用 SPA 无刷新导航（Sidebar + Content 布局），侧边栏用 Alpine.js `x-data="sidebarApp()"` 管理展开/折叠和高亮状态。

**侧边栏导航结构（5组17项）：**

```
┌─────────────────────────┐
│  📺 控制台              │
├─────────────────────────┤
│  ▶ 主播管理             │  ← 可折叠分组
│    ├── 主播配置         │  /admin/anchors
│    ├── 直播详情         │  /admin/anchor-income
│    └── 薪酬计算         │  /admin/anchor-salary
│  ▶ 排班管理             │  ← 可折叠分组
│    ├── 排班方案         │  /admin/schedules
│    ├── 排班日历         │  /admin/schedule-calendar
│    └── 排班日志         │  /admin/schedule-log
│  ▶ 直播计划             │  ← 可折叠分组
│    ├── 直播账户         │  /admin/ad-accounts
│    ├── 投流计划         │  /admin/ad-plans
│    └── 线索管理         │  /admin/leads
│  ▶ 拓展配置             │  ← 可折叠分组
│    ├── 成单管理         │  /admin/deals
│    └── 招生团队         │  /admin/recruit-teams
│  ▶ 系统配置             │  ← 可折叠分组
│    ├── AI配置           │  /admin/ai
│    ├── 邮箱配置         │  /admin/email
│    ├── 系统设置         │  /admin/settings
│    ├── API线索账号      │  /admin/clue-configs

├─────────────────────────┤
│  🚪 退出登录            │
└─────────────────────────┘
```

**SPA导航机制：**
- 点击菜单项 → `loadContent(url)` → fetch页面 → 替换 `<main>` 内容 → 重执行脚本 → `Alpine.initTree(main)`
- 路径变化 → 更新 `_path` 响应式属性 → Alpine 自动重算 `isActive()` 高亮绑定 → `_autoExpand()` 自动展开分组
- 浏览器前进/后退 → `popstate` 事件 → `loadContent(location.pathname, true)`（不 pushState）
- 网络异常时降级为 `location.href = url` 整页刷新

### 2.3 成单管理 `/admin/deals`

- 列表展示已标记的成单记录
- [新增成单] 按钮 → 弹窗：选择场次、选择线索（可选）、客户名称、金额、时间、员工、备注
- 支持编辑和删除

### 2.4 主播配置 `/admin/anchors`

- 主播列表（名称、创建时间）
- [新增主播] 按钮
- 场次关联：在场次详情中可关联主播（多选）

### 2.5 AI配置 `/admin/ai`

- API_KEY 输入框（密码类型）
- base_url 输入框（如 https://api.openai.com/v1）
- model_id 输入框（如 gpt-4o）
- 系统提示词 多行文本框
- 用户提示词模板 多行文本框（支持变量如 {{session_data}}）
- [保存] [测试连接] 按钮

### 2.6 邮箱配置 `/admin/email`

- SMTP服务器 输入框
- SMTP端口 输入框
- 发件人邮箱 输入框
- 授权码 输入框（密码类型）
- 收件人列表（可添加多个邮箱）
- [保存] [发送测试邮件] 按钮

### 2.7 系统设置 `/admin/settings`

- 修改管理员密码
- 当前数据库大小
- 最近同步时间

### 4.3 成单管理弹窗字段细节

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 关联场次 | 下拉选择 | 否 | 搜索框支持按日期筛选 |
| 关联线索 | 下拉选择 | 否 | 仅显示所选场次下的线索 |

| 成交金额 | 数字输入 | 是 | 单位：元，保留2位小数 |
| 成交时间 | 日期选择器 | 是 | 默认今天 |

| 备注 | 多行文本 | 否 | 最长200字符 |

- 选择"关联场次"后，"关联线索"下拉框才可用

- 编辑时所有字段可修改；删除需二次确认
