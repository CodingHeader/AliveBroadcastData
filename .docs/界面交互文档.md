### 2.3 成单管理 `/admin/deals`

- 列表展示已标记的成单记录
- [新增成单] 按钮 → 弹窗：选择场次、选择线索（可选）、客户名称、金额、时间、员工、备注
- 支持编辑和删除

### 2.4 主播配置 `/admin/anchors`

- 主播列表（名称、创建时间）
- [新增主播] 按钮
- 场次关联：在场次详情中可关联主播（多选）

### 2.5 AI配置 `/admin/ai`

- API_KEY 输入框（密码类型）
- base_url 输入框（如 https://api.openai.com/v1）
- model_id 输入框（如 gpt-4o）
- 系统提示词 多行文本框
- 用户提示词模板 多行文本框（支持变量如 {{session_data}}）
- [保存] [测试连接] 按钮

### 2.6 邮箱配置 `/admin/email`

- SMTP服务器 输入框
- SMTP端口 输入框
- 发件人邮箱 输入框
- 授权码 输入框（密码类型）
- 收件人列表（可添加多个邮箱）
- [保存] [发送测试邮件] 按钮

### 2.7 系统设置 `/admin/settings`

- 修改管理员密码
- 当前数据库大小
- 最近同步时间

## 四、补充交互说明

### 4.1 线索质量分布图（线索总览页）

在线索总览页顶部筛选栏下方，展示两个饼图：

```
┌──────────────────────────┬──────────────────────────┐
│  线索来源分布（饼图）      │  线索城市TOP10（饼图）     │
│  • 付费线索 65%           │  • 重庆 25%               │
│  • 自然线索 35%           │  • 成都 18%               │
│                          │  • 其他 ...               │
└──────────────────────────┴──────────────────────────┘
```

- 付费/自然判断规则：ad_account 不为空且不为"--" → 付费
- 点击饼图扇区可联动筛选下方列表

### 4.3 成单管理弹窗字段细节

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 关联场次 | 下拉选择 | 否 | 搜索框支持按日期筛选 |
| 关联线索 | 下拉选择 | 否 | 仅显示所选场次下的线索 |
| 客户姓名 | 文本输入 | 是 | 最长50字符 |
| 成交金额 | 数字输入 | 是 | 单位：元，保留2位小数 |
| 成交时间 | 日期选择器 | 是 | 默认今天 |
| 负责员工 | 文本输入 | 否 | |
| 备注 | 多行文本 | 否 | 最长200字符 |

**交互逻辑：**
- 选择"关联场次"后，"关联线索"下拉框才可用
- 保存后自动更新 leads 表对应记录的 is_deal=true
- 编辑时所有字段可修改；删除需二次确认

### 4.4 移动端适配说明

| 断点 | 布局调整 |
|------|----------|
| ≥1024px | 完整桌面布局 |
| 768-1023px | 看板卡片2列、侧边栏折叠 |
| <768px | 单列布局、导航变为底部Tab、图表宽度100%、表格横向滚动 |

- TailwindCSS响应式class实现（sm:/md:/lg:前缀）
- 后台管理页在手机端不做优化（管理员使用PC）
- 邮件模板已用max-width:700px约束，手机邮件客户端自适应
